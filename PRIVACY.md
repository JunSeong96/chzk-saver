# 치직 세이버 개인정보처리방침

시행일: 2026년 6월 17일

치직 세이버는 치지직 다시보기와 클립 링크를 MP4 파일로 저장할 수 있도록 돕는 브라우저 확장 프로그램입니다.

## 1. 수집 또는 처리하는 데이터

치직 세이버는 기능 제공을 위해 다음 데이터를 브라우저 내에서 처리할 수 있습니다.

- 사용자가 직접 입력한 치지직 다시보기 또는 클립 URL
- 해당 URL에서 불러온 영상 제목, 썸네일, 재생 목록, 화질 정보, 영상 세그먼트 등 웹사이트 콘텐츠
- 편집기에 추가한 영상 목록, 선택한 저장 구간, 선택한 화질, 마지막으로 입력한 URL 등 확장 프로그램 사용 상태

치직 세이버는 이름, 이메일 주소, 비밀번호, 결제 정보, 건강 정보, 위치 정보, 개인적인 커뮤니케이션, 전체 웹 방문 기록을 수집하지 않습니다.

## 2. 데이터 사용 목적

처리되는 데이터는 다음 기능을 제공하기 위해서만 사용됩니다.

- 치지직 다시보기 및 클립 정보 불러오기
- 영상 미리보기
- 저장 구간 선택
- 화질 선택
- MP4 파일 생성 및 브라우저 기본 다운로드 폴더 저장
- 편집기 항목 및 다운로드 작업 상태 유지

## 3. 데이터 저장

일부 설정과 작업 상태는 사용자의 브라우저 로컬 저장소에 저장될 수 있습니다. 이 데이터는 확장 프로그램 기능 제공을 위한 목적으로만 사용됩니다.

## 4. 데이터 공유

치직 세이버는 수집 또는 처리한 데이터를 개발자 서버, 광고 네트워크, 데이터 브로커 또는 기타 제3자에게 판매하거나 공유하지 않습니다.

확장 프로그램은 별도의 자체 서버를 운영하지 않습니다. 영상 정보와 미디어 데이터를 불러오기 위한 요청은 사용자가 입력한 치지직 링크 처리에 필요한 치지직, Naver API 및 관련 CDN 도메인으로만 전송됩니다.

## 5. 원격 코드

치직 세이버는 원격 코드를 사용하지 않습니다. 확장에서 실행되는 JavaScript는 모두 확장 프로그램 패키지에 포함되어 있습니다. 네트워크 요청은 영상 정보, 썸네일, 재생 목록, 영상 세그먼트 같은 데이터와 미디어를 가져오기 위한 것이며 외부 JavaScript 또는 Wasm을 불러와 실행하지 않습니다.

## 6. 데이터 보안

치직 세이버는 브라우저 확장 프로그램 권한을 기능 제공에 필요한 범위로 제한하며, 사용자의 데이터를 외부 서버에 저장하지 않습니다.

## 7. Chrome Web Store 사용자 데이터 정책 준수

치직 세이버의 데이터 사용은 Chrome Web Store 사용자 데이터 정책 및 Limited Use 요구사항을 준수합니다. 처리되는 데이터는 확장 프로그램의 단일 목적 기능을 제공하기 위해서만 사용됩니다.

## 8. 문의

개인정보처리방침에 대한 문의는 확장 프로그램 배포 페이지 또는 개발자 연락처를 통해 문의할 수 있습니다.
