# 치직 세이버

치직 세이버는 치지직 다시보기와 클립을 MP4 파일로 저장할 수 있는 확장 프로그램입니다.

## 주요 기능

- 치지직 다시보기 저장
- 치지직 클립 저장
- 여러 링크를 한 번에 편집기에 추가
- 원하는 화질을 선택해 MP4로 저장
- 영상 전체 저장 또는 원하는 구간만 선택 저장
- 구간 선택을 위한 타임라인 편집 UI 제공
- 구간 선택 중 미니 썸네일 미리보기 지원
- 편집기에 추가한 여러 영상을 목록으로 관리
- 여러 다운로드 작업을 동시에 처리
- 다운로드 진행률, 저장 용량, 속도 표시

## 사용 방법

1. 치지직 다시보기 또는 클립 페이지에서 확장 프로그램을 엽니다.
2. "편집기에 추가" 버튼을 눌러 영상을 편집기에 추가합니다.
3. 편집기에서 저장할 영상을 선택합니다.
4. 필요한 경우 타임라인에서 저장할 구간을 지정합니다.
5. 화질을 선택한 뒤 "선택 구간 저장" 버튼을 누릅니다.

## 참고 사항

- 치직 세이버는 네이버/치지직과 관계없는 비공식 오픈소스 확장 프로그램입니다.
- 저장 가능한 영상은 치지직에서 정상적으로 접근 가능한 다시보기와 클립에 한정됩니다.
- 비공개, 삭제됨, 접근 제한, 성인 인증 등으로 사용자가 접근할 수 없는 영상은 저장되지 않을 수 있습니다.
- 다운로드 속도는 사용자 네트워크 상태, 브라우저 환경, 치지직/CDN 상태에 따라 달라질 수 있습니다.
- 저장 파일은 브라우저의 기본 다운로드 폴더에 저장됩니다.
